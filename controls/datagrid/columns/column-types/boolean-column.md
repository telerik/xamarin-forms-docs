# DataGridBooleanColumn #

The **DataGridBooleanColumn** is used to represent boolean values. It uses Switch control to edit its values in EditMode.

## Example

	   <telerikGrid:DataGridBooleanColumn PropertyName="IsChampion" 
                                          HeaderText="Champion?"/>
